# ReadMe
## Igro_test A
Скрипт подключения к API курса валют и записи данных в БД
API брал currencylayer. HR`у я выслал свой токен, чтобы лишний раз вам не создавать. Вставьте его в .env

## При необходимости:
### Запустите виртуальное окружение:
```
python -m venv venv
```
### Установите последнюю версию pip
```
python.exe -m pip install --upgrade pip
```

## Установите зависимости и библиотеки
```
pip install --no-cache-dir -r requirements.txt
```

# БД по желанию 
```
docker compose up --build
```
### Или запустите сервер на системе

# Данные для БД и свой токен от API - вписывать в .env файл

# ПО РАБОТЕ
## После успешного запуска БД просто запускайте файл main. В консоли заприсит указать период вызовов API и все - скрипт раз в N время его вызывает. Запускал на 8-9 часов (я спал :D) с промежутком 20 минут - все работало отлично. Да и эксцессов быть не должно
## Логгирование также настроено. Логи сохраняются в logs/logs с указанием времени запуска скрипта. Много не делал - запуск, ошибки, запросы и все.
## SQL скрипты находятся в sql_scipts. Заданный заданием скрипт - query_stories.sql. Помимо него тал лежит скрипт для добавления данных, используемый кодом, и скрипт достающий последнюю выгрузку из БД. JOIN в запросах просто дастает title из таблицы currency
